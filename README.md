# UltiSnips
Vim UltiSnips Snippets for Rust Programming Language

# Installation

```
Put .snippets file into ~/.vim/ultisnips/UltiSnips/

or
~/.vim/UltiSnips
```

# 각 언어 들어가서 

- 누르고 정리해 주면 된다.

```
:UltiSnipsEdit
```


```
snippet 이니시에이팅Initiating  "패턴 써주기"
하고 싶은거
endsnippet
```
